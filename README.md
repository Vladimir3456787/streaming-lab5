# Fraud Detection System - Вариант 5

## Описание проекта
Система обнаружения мошеннических транзакций в реальном времени для e-commerce платформы.

## Детали варианта
- **Вариант**: 5 (последняя цифра зачетной книжки)
- **Бизнес-сценарий**: Fraud Detection  
- **Уникальное требование**: P95 задержка < 75 мс
- **Технология**: Apache Flink

## Архитектура
Система использует true streaming обработку с Apache Flink для достижения низкой задержки.

### Компоненты:
- **Kafka**: Прием и буферизация транзакций
- **Flink**: Потоковая обработка и анализ
- **Redis**: Feature store и кэширование
- **ML Model Server**: Обнаружение мошенничества
- **Prometheus+Grafana**: Мониторинг
